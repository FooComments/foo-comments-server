**Requirements**
- Node.js
- NPM
- MongoDB


Clone the repository and install dependencies:

```
$ git clone https://github.com/FooComments/foo-comments-server.git
$ cd foo-comments-server
$ npm install
```

Copy and edit the `env` file:

```
cp .env.example .env
vim .env
```

Run the server:

```
npm run server
```
